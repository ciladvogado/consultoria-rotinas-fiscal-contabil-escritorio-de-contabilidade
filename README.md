# Consultoria em Rotinas Fiscais e Contábeis

**Versão do Projeto:** 2.1
**Última atualização:** 06/03/2026

Planejamento estratégico modular para uma empresa de consultoria especializada em **planejamento e implantação de rotinas fiscais e contábeis**, atendendo dois perfis de clientes:

- **Perfil 1 — Empresas particulares** que desejam internalizar o departamento de contabilidade.
- **Perfil 2 — Escritórios de contabilidade** que desejam padronizar a prestação de serviços para múltiplos clientes.

## Portal Web

O projeto inclui um portal web full-stack para visualização do planejamento e gestão de mentorados. O código-fonte está na pasta [`portal-web/`](portal-web/).

| Recurso | Descrição |
|---------|-----------|
| [README do Portal](portal-web/README.md) | Documentação técnica do portal web |
| [Estratégia de Backup](portal-web/BACKUP.md) | Backup e restauração do banco de dados |

O portal está hospedado na plataforma Manus e inclui: dashboard com estatísticas, visualização dos 9 módulos, timeline da mentoria, gestão de mentorados com perfil individual, sistema de 5 papéis (Mentor, Mentorado, Supervisor, Revisor, Operador) e autenticação.
